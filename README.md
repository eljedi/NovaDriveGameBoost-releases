# NovaDrive & GameBoost

Outil de diagnostic, nettoyage et optimisation pour Windows 10/11.

## 📥 Téléchargement

👉 [Télécharger la dernière version (.exe)](https://github.com/eljedi/NovaDriveGameBoost-releases/releases/latest)

## ✨ Fonctionnalités

- 📊 Tableau de bord & Moniteur (CPU, GPU, RAM, températures)
- 🏎️ Benchmark
- 🔧 Vérification des pilotes & runtimes
- 🚀 GameBoost (profils Safe / Performance Max / Custom)
- 🧹 Nettoyeur (caches jeux, GPU, Windows)
- 🌐 Diagnostic réseau
- 🎮 Détection de goulots d'étranglement (bottleneck)
- 🩺 Diagnostic matériel complet avec export de rapport

- Windows 10 (build 19041+) ou Windows 11
